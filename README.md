# Learn Git

> My personal Git learning journey, documented for myself and anyone getting started with Git.

## About

I'm currently learning Backend Development, and one of the first skills every backend developer should master is Git.

This repository contains the notes I take while learning Git. Instead of keeping them on my computer, I decided to make them public in case they can help someone else who is just getting started.

The goal of this repository is not to replace the official Git documentation. Instead, it serves as a collection of beginner-friendly notes, examples, and explanations that helped me understand Git.

If you notice an error or think an explanation could be improved, feel free to open an Issue or submit a Pull Request. Every suggestion is welcome.

---

## Learning Resource

Most of these notes are based on Colt Steele's Git course.

Many thanks to him for creating such beginner-friendly content.
